# Profile (template)

Copy this file to `profile.md` for LOCAL runs, or paste its contents into the
GitHub Secret named `PROFILE_BLURB` for the scheduled cloud runs.
`profile.md` is git-ignored so your details never land in a public repo.

The text below is fed to the referral-message generator. Write it in the first
person, a single tight paragraph plus a couple of bullets — whatever reads well
in a short LinkedIn/email referral ask.

---

I'm a <degree> graduating <month year> with <N years> of experience in <domain>.
Core skills: <skill, skill, skill>. I've built <one or two flagship projects>.
I'm targeting <role types> roles in <location>. Work authorization: <status>.

Highlights:
- <quantified win #1>
- <quantified win #2>
